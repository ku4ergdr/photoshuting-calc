# Калькулятор стоимости фотосъёмки

Простой веб-калькулятор помогает фотографу рассчитывать стоимость заказа с учётом разных этапов работы, коэффициента сложности и дополнительных условий.

## Возможности
- ввод часов на дорогу, съёмку и обработку;
- умножение часов на настраиваемые ставки для каждого вида работ;
- поле для ввода коэффициента сложности фотосета;
- дополнительные коэффициенты за съёмку в выходной день и в раннее/позднее время (+10% к сложности каждый);
- сравнение полученной суммы с желаемым ежемесячным доходом;
- отдельный интерфейс администратора для настройки ставок и базового коэффициента (сохранение значений в `localStorage`).

## Использование
1. Откройте файл `index.html` в браузере.
2. Укажите желаемый доход в месяц и параметры заказа.
3. Нажмите «Рассчитать стоимость», чтобы увидеть итоговую сумму и долю от цели.
4. Для изменения ставок раскройте блок «Настройки администратора».

> Настройки администратора сохраняются локально в браузере, поэтому у каждого пользователя они свои.
